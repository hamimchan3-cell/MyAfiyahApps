**Role:**
You are a professional UI/UX Designer specializing in mobile health applications. Design a simple, clean, and user-friendly **First Aid Learning Mobile App (Android APK)** for the general public.

**Design Goal:**
Create an intuitive mobile application that teaches basic first aid and helps users respond quickly during emergencies. The interface must be easy to understand for users with little or no medical knowledge.

**Target Users:**
General public, students, families, and individuals with no medical background.

**Design Style:**

* Clean and minimal UI
* Calm medical color palette (white, soft blue, light green, red for emergency actions)
* Large readable typography
* Clear icons and illustrations
* Simple navigation suitable for stressful emergency situations
* Touch-friendly buttons

**Platform:**
Android mobile app

**Primary Navigation:**
Bottom navigation bar with 4 main tabs:

1. Home
2. First Aid Tools
3. Treatment Guides
4. Nearby Hospitals

---

**Key Features & Screens**

**1. Welcome / Onboarding Screen**

* App logo
* Short message: “Learn First Aid. Save Lives.”
* Simple illustrations about helping people
* Button: “Start Learning”

---

**2. Home Dashboard**
Main menu cards for quick access:

* First Aid Tools
* Treatment Guides
* Emergency Help
* Find Nearby Hospital

Include:

* Friendly greeting
* Emergency quick access button (red)

---

**3. First Aid Tools Feature**
Interactive list of basic medical tools with icons:

* Bandage
* Antiseptic
* Gauze
* Medical Gloves
* Scissors
* Thermometer
* First Aid Kit

Each item opens a **detail screen** showing:

* Tool image
* Tool name
* Short explanation
* How and when to use it
* Simple visual illustration

---

**4. Treatment Guides**
Step-by-step guides for common injuries:

* Cuts
* Burns
* Sprains
* Nosebleeds
* Minor bleeding
* Fainting

Each guide should include:

* Simple step list
* Visual illustrations
* Warning signs for when to seek medical help

Example structure:

1. Assess the situation
2. Clean the wound
3. Apply treatment
4. Monitor condition

---

**5. Emergency Help Screen**
Quick emergency actions:

* Call emergency services
* Quick first aid tips
* Big red emergency button

---

**6. Nearby Hospitals Feature**
Map-based screen that:

* Shows nearby hospitals or clinics
* Displays distance
* Button: “Navigate”
* Uses GPS location

Include:

* Map view
* List view of nearby hospitals

---

**UX Requirements**

* Easy navigation in stressful situations
* Large buttons and icons
* Clear step-by-step instructions
* Minimal text
* Accessible for beginners
* Fast access to emergency help

---

**Design Deliverables**
Create the following screens:

1. Splash Screen
2. Onboarding
3. Home Dashboard
4. First Aid Tools List
5. Tool Detail Page
6. Treatment Guides List
7. Step-by-Step Treatment Page
8. Emergency Help Page
9. Nearby Hospitals Map Screen

Design the interface as a **modern mobile UI prototype with consistent colors, icons, and spacing.**
